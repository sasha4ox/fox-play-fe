# Dashboard hero banners

Place promo images here (PNG/JPG/WebP). Update the slide list in:

`src/components/DashboardHeroBanner/heroBanners.js`

Suggested workflow: export assets from your design folder (e.g. Desktop `FoxyPlayBanners`) as `banner-01.png`, `banner-02.png`, … and reference them in `HERO_BANNER_SLIDES`.
