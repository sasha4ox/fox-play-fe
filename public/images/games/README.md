# Game images

Place game cover/banner images here for the games selection page. Images are shown on each game card.

**Naming convention:**
- Use the game name as filename (lowercase, spaces → hyphens).
- Supported formats: PNG, JPG, WebP.

**Examples:**
- `lineage-2.png` – for game "Lineage 2"
- `cs2.png` – for game "CS2"
- `dota.png` – for game "Dota 2"

If a game has no image here and no `imageUrl` in the database, `default.png` is used as fallback.
